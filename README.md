# Notisave

Notisave is an Android application designed to capture and save your device's notifications, allowing you to view and manage them later.

## Features
- **Notification History**: Automatically saves incoming notifications so you never miss an important message.
- **Detailed View**: View detailed information about past notifications.
- **Modern UI**: Clean and intuitive user interface built entirely with Jetpack Compose.
- **Local Storage**: Securely stores your notifications locally using a Room database.

## Technologies Used
- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Database:** Room
- **Build System:** Gradle

## Getting Started

To build and run the project locally:

1. Clone this repository.
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the app on an Android emulator or a physical Android device.
